# Terminal Audio Oscilloscope 📊🎵

Визуализируйте ваш аудиопоток прямо в терминале! Плавный, высокооптимизированный осциллограф на Python, использующий шрифты Брайля для графики высокой четкости и поддерживающий векторные фигуры Лиссажу.

## ✨ Особенности
* **3 режима отображения:**
  * `XY MODE` — классический аналоговый клубок (фигуры Лиссажу) для стерео-сигнала.
  * `SINGLE CHANNEL WAVE` — объединенная моно-осциллограмма.
  * `DUAL CHANNEL WAVE` — раздельный двухканальный просмотр (левый/правый канал).
* **Умная жирность линий:** Тонкие и аккуратные линии для режима Лиссажу, чтобы не терять детализацию, и сочные, плотные, жирные линии для режимов волны.
* **Высокая оптимизация:** Отрисовка через быстрый алгоритм Брезенхема и векторные маски NumPy для работы на стабильных 50 FPS без нагрузки на процессор.
* **Настоящая плавность:** Настраиваемый коэффициент инерции луча (smoothing) для мягкого аналогового движения.
* **Совместимость:** Отлично работает в большинстве современных терминалов Linux.
* **100% чистый Python!**
